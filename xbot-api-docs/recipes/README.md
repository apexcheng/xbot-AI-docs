# 实战方案

`recipes/` 目录存放基于影刀 xbot API 的业务场景组合方案。

## 与 `docs/` 的区别

| 目录 | 定位 |
|---|---|
| `docs/` | API 能力说明——每个模块有哪些方法、参数、注意事项 |
| `recipes/` | 实际业务场景——如何组合多个 API 完成一个完整任务 |

## 当前方案

| 方案 | 文档 | 说明 |
|---|---|---|
| 价格监控 | [price-monitoring.md](price-monitoring.md) | 定时采集商品价格、判断异常、汇总通知 |

## 新增方案规范

1. 每篇方案应包含：业务背景、推荐分工、关键步骤、注意事项。
2. 方案中引用的 API，应链接到 `docs/` 对应文档。
3. 不确定的实现细节，标注 `"需运行验证"`。
