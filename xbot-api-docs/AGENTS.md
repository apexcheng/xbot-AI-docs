# AGENTS.md

本目录是影刀 xbot API 知识库子目录，不作为 Agent 开发主目录。

实际影刀项目开发时，Agent 应在真实影刀项目根目录工作；本目录只用于查阅 API 文档、示例和排错资料。

当前知识库仓库的稳定 Agent 规则入口见：`../AGENTS.md`

## 防漏硬规则

如果 Agent 只读取到本文件，也必须遵守以下规则：

1. 写影刀编码版代码前，先查本目录下的 `docs/`。
2. 不确定 API 行为时，不要猜，标注“需运行验证”。
3. 涉及浏览器、URL、网页业务功能时，除非用户明确要求，不要使用 `requests`、`httpx`、`aiohttp`、`urllib` 或其他网络相关 Python 库；只使用 `xbot.web` 及其浏览器对象能力。
4. 市场指令参数不明确时，先查 `docs/debug/market-extension-source.md`。
5. 只修改与当前任务直接相关的内容，不做无关重构，不改动无关文件、格式、命名或结构。
6. 不要提交影刀运行包源码、反编译源码、敏感路径、账号信息或 token。
7. 不要编造不存在的功能、命令、目录、文件或 API。

高频稳定性约定：不要根据界面中文选项猜编码版参数；能用元素操作就不要优先坐标点击；中文或长文本输入优先考虑剪贴板输入；表格大量写入优先考虑二维数组批量写入。

## 文档维护提示

新增或补充 API 文档时，优先放到 `docs/`；可运行代码片段放到 `examples/`。

每篇 API 文档尽量包含：

- 适用场景
- 常用方法
- 最小示例
- 注意事项
- 排错速查
